# Ciencia-de-datos
## Teorema del límite central
__Definición__: Si $\mathbf{\bar{X}}$ es la media de una muestra aleatoria de tamaño _n_, tomada de una población con media $\mu$ y varianza finita $\sigma^2$, entonces la forma límite de la distribución de
$$Z = \frac{\bar{X} - \mu}{\sigma / \sqrt{n}}$$
a medida que $n \to \infty$, es la distribución normal estandar _n_(z;0;1).
